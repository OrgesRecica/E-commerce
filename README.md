# E-commerce

Full-stack e-commerce application.

- **frontend/** — React + Vite + Tailwind + Redux + React Query + Axios
- **backend/** — Node.js + Express + MongoDB (Mongoose) + JWT + Stripe + Cloudinary + Nodemailer

## Quick start

```bash
# backend
cd backend
cp .env.example .env
npm install
npm run dev

# frontend
cd frontend
cp .env.example .env
npm install
npm run dev
```
